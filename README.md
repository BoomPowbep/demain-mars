## Installation

First, clone the repository
```
git clone https://github.com/BoomPowbep/demain-mars.git
```

Then, install dependencies
```
npm install
```

## Run
### Encore
#### Watch (dev)
```
npm run encore.watch
```

#### Build 
```
npm run encore.prod
```

### HTTP-Server
```
npm run serve
```
